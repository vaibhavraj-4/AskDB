DBInsight Documentation

Welcome to the official documentation of DBInsight. This guide will help you install, configure, and use DBInsight effectively.

Getting Started

Follow these steps to set up DBInsight:

  1. Install Python 3.9 or higher
  2. Clone the repository: 
    git clone https://github.com/your-repo/dbinsight.git
  3. Install dependencies: 
    pip install -r requirements.txt
  4. Run the Flask server: 
    flask run
  5. Open in browser: 
    http://127.0.0.1:5000

Usage Guide

Once DBInsight is running, you can interact with your database using natural language queries:

  • Connect: Use the connection modal to enter DB credentials.
  • Ask: Type queries like "Show me top 10 customers".
  • Translate: DBInsight automatically converts text into SQL.
  • View: Results are shown as tables and charts.
  • Export: Download data in CSV format.

API Reference

POST /api/query

Description: Executes a natural language query and returns results.

Request Example:

{
  "query": "Show top 5 products"
}

Response Example:

{
  "status": "success",
  "data": [
    {"product": "Laptop", "sales": 1200},
    {"product": "Phone", "sales": 980}
  ]
}

FAQ

1. Which databases are supported?

Currently MySQL, PostgreSQL, and SQLite. Support for more is coming.

2. How can I export results?

Use the "Export CSV" button available in the results section.

3. My query failed, what do I do?

Check if database credentials are correct and the SQL translation is valid.
